阶段性总结（知识库存档）项目名称：超级个体 BCI 中枢 - Muse S Athena 协议攻克阶段日期：2026-03-03核心贡献：首次在 React Native 环境下完整复刻了 Muse S Athena 芯片的非公开 BLE 通讯协议。1. 核心问题点 (Pain Points)硬件锁死：使用旧版指令（如 p21, s, d）时，硬件报告通道数为 0 ("ch":[0,0])。指令错误：发送长度不匹配的指令会导致硬件返回 rc:69。推流屏障：Athena 芯片需要特定的握手序列才能开启数据闸门。2. 关键突破 (Key Breakthroughs)双发 dc001 脉冲：发现必须连续发送两次 dc001 指令（Base64: BmRjMDAxCg==），且中间配合 L1 指令同步，才能激活传感器。12-bit 多路复用解压：掌握了 3 字节携带 2 个 采样点的位移算法：$Sample1 = (Byte0 \ll 4) | (Byte1 \gg 4)$$Sample2 = ((Byte1 \& 0x0F) \ll 8) | Byte2$。物理量还原：通过公式 $(Raw - 2048) \times 0.475$ 将原始整数还原为微伏电压。3. 最终成功架构前端：react-native-ble-plx 负责物理连接与指令下发 + 127.0.0.1 WebSocket 转发。后端：FastAPI 接收 Base64 原始包，执行位移解包逻辑。
