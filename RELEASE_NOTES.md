# MuseApp Release Notes - Version 1.0.0 "Panorama Edition"

## 概述
本版本（V1.0.0）标志着 MuseApp 核心功能的全面完成。我们构建了一个稳定、高效且具备深度生物反馈能力的 Muse 脑电采集与冥想辅助平台。

---

## 核心功能 Panoramic View

### 1. 稳定性保障与后台架构 (Stability & Architecture)
- **前后端解耦**: 采用 React Context API 实现全局设备状态管理，UI 与逻辑完全分离。
- **三重后台锁**: 集成了 `Foreground Service` (前台服务)、`Keep Awake` (常亮锁) 与 `Silent Audio` (静音音频轨)，确保在 Android 锁屏或应用后台时，脑电数据采集不会中断。
- **被动监听机制**: 优化了 BLE 连接协议，支持在不干扰官方 App 运行的情况下被动接收 Muse 原始数据流。

### 2. 生物信号处理 (Bio-Signal Intelligence)
- **实时信号质量检测**: 实现了针对 TP9/AF7/AF8/TP10 四通道的智能阻抗评估，算法经过阈值优化，在佩戴初期即可获得准确反馈。
- **心率 (BPM) 监测**: 通过红外 PPG 通道实时计算心率，支持冥想过程中的心脏自主神经系统评估。
- **频域分析**: 实时提取 Theta/Alpha 波段能量，生成脑波趋势图。

### 3. 冥想与睡眠增强 (Meditation & Sleep Experience)
- **双层音频系统**: 结合 `TrackPlayer`（音乐）与 `expo-av`（环境音），提供沉浸式音频体验。
- **智能降噪/淡出**: 首次引入“入睡检测算法”，当检测到 Theta 波占比升高（进入瞌睡态）时，自动线性降低播放音量，辅助用户顺滑入睡。
- **睡眠功耗优化**: 预设 p1034 低功耗指令集支持，延长采集续航。

### 4. 数据管理与安全 (Data Sovereignty)
- **灾备自动保存**: 采用每 2 分钟缓冲区强行落板机制，防止手机闪退或掉电导致的数据丢失。
- **零依赖导出**: 使用 `fflate` 实现纯 JS 侧的 Zip 压缩打包，支持一键导出所有采集日志至社交软件或云端。
- **自动清理**: 具备日志自动老化管理机制，防止应用过度占用手机存储空间。
